# State Files

This folder contains GitHub-facing status artifacts for the integrated dashboard.

- `haver_status.json`: latest run or preflight status
- `haver_events.jsonl`: append-only run event stream
- `haver_latest_failure.json`: latest failure record for the dashboard
- `haver_failures.jsonl`: append-only failure-only history stream

These files are generated by `main.py` and `scripts/haver_preflight.py`.
